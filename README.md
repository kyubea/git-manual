# Git Workflow Guide for CS Students

[![Live Site](https://img.shields.io/badge/Live%20Site-GitHub%20Pages-blue)](https://kyubea.github.io/git-manual/)
[![Version](https://img.shields.io/badge/Version-1.0-green)](https://github.com/kyubea/git-manual)

A comprehensive, student-friendly guide to Git and version control designed specifically for computer science students. This guide covers everything from installation to collaboration, with a focus on practical workflows used in academic settings. This manual was created for 
EH-301's Fall 2025 course as a submission for the final project.

## What's Covered

- **Getting Started**: Installation, configuration, and SSH key setup
- **Daily Workflow**: Essential commands (clone, add, commit, push, pull)
- **Branching**: Creating, switching, merging, and managing branches
- **Merge Conflicts**: Understanding and resolving conflicts step-by-step
- **Collaboration**: Group workflows, pull requests, and best practices
- **Troubleshooting**: Common errors, recovery strategies, and .gitignore usage
